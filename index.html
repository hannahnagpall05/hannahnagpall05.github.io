About

The discipline of sport,
applied to engineering.

I'm Hannah — an Honors CS student at Texas A&M University–Kingsville with a focus on backend systems, machine learning, and applied research. I started my CS path at Alabama A&M University while competing as a full-scholarship NCAA Division I tennis player, then transferred to TAMUK where I've maintained a 3.92 GPA and built a research portfolio that spans spectroscopy, network dynamics, and health prediction.

I work in Java, Python, C++, SQL, HTML, and CSS, with production experience in Spring Boot microservices and ML pipelines using scikit-learn, SciPy, and pandas. I'm currently a co-author on two papers under IEEE review, and I run a live web platform for college tennis placement.

Long-term, I'm aiming for a top research-university Master's program where I can push the boundary between systems engineering and applied science.
